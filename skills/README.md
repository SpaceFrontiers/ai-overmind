# Skills

This directory contains example Agent Skills organized by category. Each skill demonstrates different patterns and capabilities.

## Categories

### Creative & Design
Skills for creative tasks, visual design, and content creation.
- [graphic-designer](./creative-design/graphic-designer/) - Generate design concepts and visual layouts
- [content-writer](./creative-design/content-writer/) - Create engaging content following style guidelines

### Development & Technical
Skills for software development, testing, and technical tasks.
- [code-reviewer](./development-technical/code-reviewer/) - Review code for quality and best practices
- [test-automation](./development-technical/test-automation/) - Generate automated tests for web applications
- [api-documenter](./development-technical/api-documenter/) - Create comprehensive API documentation

### Enterprise & Communication
Skills for business communication, branding, and professional workflows.
- [email-composer](./enterprise-communication/email-composer/) - Draft professional emails with proper tone
- [meeting-notes](./enterprise-communication/meeting-notes/) - Structure and format meeting notes
- [brand-guidelines](./enterprise-communication/brand-guidelines/) - Apply brand guidelines to content

### Personal Productivity
Skills for personal organization, planning, and efficiency.
- [task-organizer](./personal-productivity/task-organizer/) - Organize and prioritize tasks
- [research-summarizer](./personal-productivity/research-summarizer/) - Summarize research materials
- [note-taker](./personal-productivity/note-taker/) - Structure and organize notes

### Business & Finance
Skills for business operations, financial compliance, and tax planning.
- [us-tax-compliance-foreign-entities](./business-finance/us-tax-compliance-foreign-entities/) - Expert guidance on C-Corp and LLC tax compliance for foreign owners

## Using These Skills

1. Navigate to the skill folder you're interested in
2. Review the `SKILL.md` file to understand what it does
3. Download or copy the entire skill folder
4. Upload it to Claude via Claude.ai, Claude Code, or the API

## Contributing Skills

We welcome contributions! To add a new skill:

1. Choose the appropriate category or create a new one
2. Create a new directory with your skill name
3. Follow the [skill specification](../spec/README.md)
4. Use the [template](../template/SKILL.md) as a starting point
5. Submit a pull request

## Skill Complexity Levels

- **Basic**: Simple skills with just `SKILL.md`
- **Intermediate**: Skills with additional reference files
- **Advanced**: Skills with multiple files, resources, and scripts

Browse through skills of different complexity levels to learn different patterns.
